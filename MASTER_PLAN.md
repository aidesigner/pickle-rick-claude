# MASTER_PLAN compatibility snapshot

Canonical master-plan bookkeeping lives in **`prds/MASTER_PLAN.md`** (live ledger +
Drain Queue) and **`prds/MASTER_PLAN-archive.md`** (shipped releases + closed findings).

This repo-root file exists only so legacy acceptance commands that still target
`MASTER_PLAN.md` have a stable path. It carries **no status snapshot** of its own —
a duplicated snapshot here drifts out of sync. Read the canonical files for current state.
